## Customization

### Update Personal Information

1. **Name and Title**: Edit the hero section in `index.html`
2. **About Section**: Update the about text in the about section
3. **Skills**: Modify the skills grid in `index.html`
4. **Projects**: Update project cards with your actual projects
5. **Contact**: Update contact information and social links

### Change Colors

Edit the CSS variables in `styles.css`:

```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    /* ... other colors */
}
```

### Add More Sections

Simply add new sections following the same pattern in `index.html` and style them in `styles.css`.
